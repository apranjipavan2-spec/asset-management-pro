# 🎯 Kalike Roles System - User Guide

## Overview

The Kalike Asset Management System includes a comprehensive role-based documentation system with 7 distinct roles and interactive navigation tools.

---

## 🎨 Interactive Roles Hub

### File: `KALIKE_ROLES_HUB.html`

An interactive visual chart showing all 7 roles organized by hierarchy level with clickable bubbles.

**How to Use:**
1. Open `KALIKE_ROLES_HUB.html` in your browser
2. See all 7 roles arranged by organizational hierarchy
3. **Click any role bubble** to view detailed guide for that role
4. Roles are color-coded and sized by importance/level

**Visual Elements:**
- **Bubbles:** Each role shown as an interactive ellipse
- **Hierarchy Levels:** Arranged from Level 6 (top) to Level 1 (bottom)
- **Connection Lines:** Show reporting/organizational relationships
- **Color Coding:** Each role has unique color for easy identification

**Access Levels Shown:**
- Level 6: Full System Access
- Level 5: Full Access
- Level 4: Strategic & Operational Access
- Level 3: Team Management Access
- Level 1: Basic User Access

---

## 📖 Comprehensive Role Guide

### File: `KALIKE_ROLE_GUIDE.html`

Complete documentation for all 7 roles with detailed features, responsibilities, and workflows.

**How to Use:**
1. Open `KALIKE_ROLE_GUIDE.html` in your browser
2. Click any **role tab** at the top to view that role's guide
3. Each guide includes:
   - Role overview and responsibilities
   - Dashboard overview
   - Core features and capabilities
   - Workflows and processes
   - Best practices
   - FAQ section
   - Getting help resources

**Linked Navigation:**
- Click a role in the Roles Hub → Auto-navigates to that role's guide
- Click tabs in the guide → Switch between roles instantly
- Each role maintains its color theme throughout

## 🔐 The 7 Roles Explained

### Level 6: Super Administrator (🔐)
- **Color:** Red (#c41e3a)
- **Access:** Full system control
- **Typical Users:** IT admins, system maintainers
- **Key Tasks:** User management, backups, security, system config

### Level 5: Executive Director (👑)
- **Color:** Gold (#d4af37)
- **Access:** Full organizational access
- **Typical Users:** CEO, Executive leadership
- **Key Tasks:** Strategy, budgets, board reporting, policy decisions

### Level 4a: Operations Manager (⚙️)
- **Color:** Orange (#ff6b35)
- **Access:** Asset management, approvals, procurement
- **Typical Users:** Operations directors, asset managers
- **Key Tasks:** Asset registry, request approval, allocation

### Level 4b: Finance Controller (💰)
- **Color:** Teal (#11998e)
- **Access:** Financial data, depreciation, reports
- **Typical Users:** Finance managers, accountants
- **Key Tasks:** Valuation, depreciation, financial reports, grants

### Level 4c: HR Manager (👥)
- **Color:** Pink (#f5576c)
- **Access:** User management, payroll, leave
- **Typical Users:** HR directors, HR specialists
- **Key Tasks:** Users, payroll, leave, performance reviews

### Level 3: Program Coordinator (📊)
- **Color:** Blue (#667eea)
- **Access:** Team-level asset management
- **Typical Users:** Managers, team leads
- **Key Tasks:** Team requests, asset allocation, approvals

### Level 1: General Staff (👤)
- **Color:** Purple (#764ba2)
- **Access:** Personal asset requests and tracking
- **Typical Users:** All employees
- **Key Tasks:** Request assets, track allocations, report issues

## 🚀 How to Access

### Option 1: Visual Navigation (Recommended for new users)
```
1. Open KALIKE_ROLES_HUB.html
2. See all roles visually
3. Click on your role
4. View comprehensive guide automatically
```

### Option 2: Direct Guide Access
```
1. Open KALIKE_ROLE_GUIDE.html directly
2. Select your role from tabs
3. Read detailed documentation
```

### Option 3: From Application
```
1. In the main app (index.html)
2. Look for "Help" or "Roles" button
3. Opens KALIKE_ROLES_HUB.html or KALIKE_ROLE_GUIDE.html
```

## 🏗️ Architecture
- **Tech Stack**: Vanilla HTML5, JavaScript (ES6+), Tailwind CSS (CDN).
- **Architecture**: Single Page Application (SPA) with Hash-based routing.
- **Persistence**: Browser `localStorage` (Mock DB).
- **Design System**: Monolith-Shadow aesthetic with Institutional Grade spacing.

## 👥 Role-Based Access
- **Asset Manager**: Master registry, fleet tracking, Kanban maintenance hub.
- **Finance Officer**: Depreciation schedules, Net Book Value (NBV) ledger, Grant & Program tracking.
- **Employee**: Asset requisition, personalized inventory, issue reporting.

## 📦 Features
- **Real-Time Dashboards**: Bento-grid stats for all roles.
- **Maintenance Workflow**: Full Kanban-style ticket management.
- **Financial Compliance**: Automated depreciation (SLM) and Grant utilization tracking.
- **Responsive Design**: Works seamlessly across all browser viewports.

## 🛠️ How to Deploy
1. Clone this repository.
2. Serve the directory using any static web server (e.g., `npx serve .`).
3. For GitHub Pages: Enable in **Settings > Pages**.
